# Houdini のショートカット

## Viewport 上

|     |                |
| --- | -------------- |
| D   | Display Option |
| S   | Select         |

## Select

### [F8]

|     |                                            |
| --- | ------------------------------------------ |
| F8  | オブジェクト選択・プリミティブ選択切り替え |
| 1   | Select Object                              |
| 2   | Select Point                               |
| 3   | Select Edges                               |
| 4   | Select Primitives                          |

## Network

|     |                       |
| --- | --------------------- |
| R   | Display/Render に指定 |
| L   | ノードを整列          |
