    Blender の各種ショートカットまとめ

    1. 一般的なもの

    1.1. ファイルを開く (Ctrl + o)

    1.2. 検索メニューを表示する (F3)

    1.3. サイドバーを表示（N）

    1.4. ツールバー表示（T）

    1.5. マウス位置にツールバー表示（Shift+Space）

    2. 表示モード

    2.1. ViewPort を分割（境目で右クリック →SplitArea）

    2.2. 編集モードを変更する(Tab)

    ObjecctMode や Edge 編集などの作業モードを切り替える

    2.3. カメラモードの切り替え（`）

    3. 選択・移動

    3.1. Shift + S カーソルの移動、アタッチ

    3.2. Grid にアタッチ (G -> Ctrl 押しながら移動)

    3.3. スナップを ON にする（Shift+Tab）

    3.4. ものすごく細かく移動（G -> Shift 押しながら移動）

    4. カメラ

    レンダリングカメラを、現在のビューカメラ位置に移動 ( Ctl + Alt + Numpad 0 )

    5. アニメーション関係

    5.1. キーを打つ（i）
